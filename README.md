# 📖 About

**SoftBuyDeals** is a modern Amazon affiliate website built with **Astro** and **Tailwind CSS v4**, designed to deliver a fast, responsive, and SEO-optimized shopping experience.

The website showcases carefully curated products across multiple categories. Visitors can browse products, view details, and purchase securely through Amazon using affiliate links.

To maximize performance, the website **does not use a client-side JavaScript framework** such as React, Vue, or Svelte. Interactive features are implemented using **vanilla JavaScript** inside Astro `<script>` tags, resulting in minimal JavaScript, faster page loads, and an improved user experience.

# 🚀 Why Astro?

SoftBuyDeals is built with Astro because it provides:

- ⚡ Near-instant page loads
- 📈 Excellent SEO performance
- 📦 Minimal JavaScript by default
- 🏗️ Static Site Generation (SSG)
- 🔒 Improved security through static deployment
- 🌍 Better Core Web Vitals
- 🧩 Component-based development
- 🎯 Interactive features only where needed using vanilla JavaScript

This architecture keeps the website lightweight, maintainable, and optimized for affiliate marketing, where search rankings and speed are essential.
