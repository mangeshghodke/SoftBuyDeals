# 📖 About

**SoftBuyDeals** is a modern affiliate marketing website built with **Astro** to provide a fast, responsive, and SEO-friendly browsing experience.

The platform showcases carefully selected software, gadgets, accessories, and everyday products. Users can explore products and purchase them securely through Amazon using affiliate links.

> **SoftBuyDeals does not sell products directly.** All purchases are completed on Amazon.

# ⚙️ Getting Started

Clone the repository

```bash
git clone https://github.com/mangeshghodke/SoftBuyDeals.git
```

Navigate into the project

```bash
cd SoftBuyDeals
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:4321
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
