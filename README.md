# 📖 About

**SoftBuyDeals** is a modern affiliate marketing website that helps users discover high-quality products, software, gadgets, and everyday essentials.

Instead of selling products directly, SoftBuyDeals curates products and redirects visitors to Amazon using affiliate links, allowing users to shop securely while supporting the project through qualifying purchases.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/mangeshghodke/SoftBuyDeals.git
```

Go to the project

```bash
cd SoftBuyDeals
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

# 🎯 How It Works

1. Browse products.
2. Read product information.
3. Click **Buy on Amazon**.
4. Get redirected to Amazon.
5. Purchase securely through Amazon.
